##### ------------------------配置Easy Code------------------------------

File -> Settings -> Other Settings -> Easy Code

直接往里面对应粘贴

用Default



##### -------------------------数据库配置-------------------------------

1. IDEA界面右侧 -> Database -> + -> Data Source -> 连接Mysql数据库
2. idea链接数据库URL后缀：  -------------->   192.168.1.242  testuser  Ceshi#@!0714
------   ?useUnicode=true&characterEncoding=utf8&serverTimezone=GMT%2B8&useSSL=false
3. Test Connection -> 连接成功
4. 点击 Schemas  取消选择 Default schema  选择 small_neighborhood
5. File -> Settings -> Languages & Frameworks -> SQL Dialects -> 两个都选择 MySQL  点下右下按钮Apply
6. 还是这个界面的 下一个选项 -> SQL Resolution Scopes -> 取消选择 All Data Source -> 选择 small_neighborhood -> 点下右下按钮Apply

数据库配置完成



##### -------------------------使用方法-------------------------------

Easy Code

1. 在 Database 中 点击需要生成的表  -右键>  EasyCode -> Generate Code
2. 随便生成个随便的位置
3. 粘贴到你项目对应的位置 controller service serviceImpl mapper mapper.xml



MyBatisCodeHelperPro

1. 在 Database 中 点击需要生成的表  -右键>  Mybatis generator

2. module 选择 `small-object`

3. 默认生成文件：

   ```html
   1. module 选择 small-object
   2. mapper 生成到 `com.shiyi.mapper`
   3. mapper xml 生成到 `mapper`
   ```

4. lombok 除了 @GetterSetter 都选

5. 配置

   ```html
   1. 不生成jdbcType
   ```

6. 点击下面的 service

   ```html
   选上 generateService
   
   包名 -----> com.shiyi.service.impl
   ```

   

7. 点击下面的 service interface

   ```html
   选上 generateServiceInterface
   
   包名 -----> com.shiyi.service
   ```

8.  生成完成 快去看看吧！！！！！！！
